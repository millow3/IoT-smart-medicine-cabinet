# IoT-smart-medicine-cabinet
An IoT-based Smart Medicine Cabinet Monitoring and Remote Caregiver Alert System using ESP32, ESP8266, Firebase Realtime Database, RFID authentication, and real-time web monitoring. Developed for TTTR3163 Internet of Things, Universiti Kebangsaan Malaysia (UKM).
